# 심플리투두
심플리투두는 인터넷 연결을 요구하지 않는 아주 간단한 할일 목록 정리 앱입니다.


## 제공 기능
- 할일 목록 생성 및 삭제
- 생성된 할일 목록에 대한 푸시 알림 제공
- 할일 완료 여부 체크 기능
